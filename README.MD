极简空气质量应用，弹指间了解空气质量指数（AQI）及6种污染物的浓度指标。

## 技术概览

* [React Native](https://facebook.github.io/react-native)强力驱动，主要在`iOS`之上调试部署。
* 空气质量数据来自[pm25.in](http://pm25.in/)开放API。
* 采用[高德地图Web服务](https://lbs.amap.com/)进行城市地理逆解析。
